# AI-Enhanced Multi-Agent Hospital Logistics System

## Overview

This project presents an Intelligent Multi-Agent System (MAS) developed in AnyLogic for autonomous hospital logistics operations. The simulation models a fleet of mobile robots responsible for transporting medical supplies, equipment, and resources within a hospital environment.

The objective is to demonstrate how multiple intelligent agents can cooperate, communicate, and make decisions in a dynamic environment while improving task allocation efficiency and resource utilization.

---

## Features

### Autonomous Robot Agents

Each robot operates as an independent intelligent agent capable of:

* Receiving tasks
* Navigating the environment
* Executing logistics operations
* Participating in task allocation strategies

### Dynamic Task Generation

Tasks are generated dynamically during simulation, creating a realistic and continuously changing workload for the robotic fleet.

### Configurable Parameters

Users can customize simulation behavior before execution through an interactive setup screen:

* Number of Robots
* Task Generation Rate
* Robot Speed
* Allocation Mode

### Multiple Allocation Strategies

The simulation supports multiple task allocation modes, allowing comparison between different coordination approaches and decision-making mechanisms.

### Interactive User Interface

The project includes:

#### Introduction Screen

Provides an overview of the system and allows users to:

* Start the simulation
* Open parameter configuration

#### Parameter Configuration Screen

Allows customization of system settings before execution.

#### Simulation Screen

Displays real-time robot operations, task allocation, and system behavior.

---

## Multi-Agent Architecture

The system consists of several interacting agents:

### Robot Agents

Responsible for task execution and movement within the environment.

### Task Generation Agent

Creates logistics tasks dynamically during simulation.

### Coordinator Agent

Manages task allocation and system coordination.

### Environment

Represents the hospital logistics workspace where agents interact and perform operations.

---

## Simulation Workflow

1. User configures simulation parameters.
2. Tasks are generated dynamically.
3. Robots receive available tasks.
4. Allocation strategy selects the most suitable robot.
5. Selected robot executes the task.
6. Performance metrics are updated continuously.
